## Overview

**MBLabs** is a CLI made with [Nest.js](https://nestjs.com/) and [Nest.js Commander](https://docs.nestjs.com/recipes/nest-commander). It performs miscellaneous tasks that do not quite fit in other projects. For instance, it:

- Houses various prompts used by our Taskfile.yml setup

## Requirements

- **[Python >3.10.0](repository.project.python)**
